---

## 🚀 Main Project

## 🎧 Spotify Clone

A web-based music streaming application inspired by Spotify.

### ✨ Features
- Dynamic folder-based song loading
- Song listing using Fetch API
- Play / Pause controls
- Song switching
- Interactive UI updates
- Audio control handling

### 🧠 Concepts Practiced
- DOM Manipulation
- Event Handling
- Fetch API
- Asynchronous JavaScript
- Dynamic Content Rendering
- Audio API usage

---

## 🧪 JavaScript Mini Projects

### 🧮 Faulty Calculator
A calculator that intentionally produces incorrect results 10% of the time to simulate logic-based randomness.

**Concepts Used:**
- Random number generation
- Conditional logic
- Mathematical operations

---

### 🔢 Factorial Finder
Calculates the factorial of a user-provided number.

**Concepts Used:**
- User input handling
- Loops
- Mathematical logic

---

### 🖼 Image Slider
A dynamic image slider with navigation controls.

**Concepts Used:**
- Event listeners
- DOM traversal
- Array indexing
- UI state management

---

### 💻 Hacking Terminal Simulation
A terminal-style animated UI that simulates a hacking environment.

**Concepts Used:**
- setTimeout()
- setInterval()
- Text animation effects
- Sequential execution logic

# 🚀 Main Project

## 🎧 Spotify Clone

> ⚠️ **Known Issue Notice**  
> The dynamic song fetching functionality is currently not working as expected due to an incomplete or unstable Fetch API implementation.  
>  
> As a result, songs may not load correctly from folders.  
>  
> This issue has been identified and will be fixed in a future update.  
>  
> All other UI components, structure, and audio control logic are implemented and functional.

A web-based music streaming application inspired by Spotify.

### ✨ Features
- Dynamic folder-based song loading (currently under debugging)
- Song listing using Fetch API
- Play / Pause controls
- Song switching
- Interactive UI
- Audio playback handling

### 🧠 Concepts Practiced
- DOM Manipulation
- Event Handling
- Fetch API
- Asynchronous JavaScript
- Dynamic Rendering
- Audio API usage

# 🧪 JavaScript Mini Projects

## 🧮 Faulty Calculator

A calculator that intentionally produces incorrect results 10% of the time to simulate logic-based randomness.

### Concepts Used:
- Random number generation
- Conditional statements
- Mathematical logic
- Edge case handling

---

## 🔢 Factorial Finder

Calculates the factorial of a user-provided number.

### Concepts Used:
- User input handling
- Loops
- Mathematical operations
- Basic algorithm logic

---

## 🖼 Image Slider

A dynamic image slider with navigation controls for switching between images.

### Concepts Used:
- Event listeners
- DOM traversal
- Array indexing
- UI state management

---

## 💻 Hacking Terminal Simulation

A terminal-style animated interface that simulates a hacking environment using timed text output.

### Concepts Used:
- setTimeout()
- setInterval()
- Sequential logic
- DOM updates
- Text animation effects
